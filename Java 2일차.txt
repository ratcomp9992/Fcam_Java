9. 실수형 배열을 매개변수로 받아서 총합을 구하여 리턴하는 메서드를 정의해보시오.(메서드명 floatTotal)
	Double floatTotal (double[] arr) {
		double sum = 0;
		
		for (int i = 0; i < arr.length; i++) {
			sum += arr[i];
		}
		return sum;
	}
